# Project Title

This is a small *personal* project to reinforce React and REST APIs

## Description

This project uses React and TMDB's API to create a simple movie database with a search function and favorites function using browser localStorage. If you want to view the current web page now, visit: https://cv0506-moviedb-project.netlify.app/

## Getting Started

### API Key

* If you would like to run this project locally, you need a TMDB API key. TMDB offers their API keys completely for free at https://www.themoviedb.org/
* Modify the api.js file in the services folder to hold your API key.

### Dependencies

* This project uses React and Vite, both of which should be in package.JSON

### Executing program

* Locally, this program runs with Vite
```
npm run dev
```
